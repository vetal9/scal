# 📈 Trading App — Binance Spot OCO

Мобильное веб-приложение для трейдинга на Binance Spot.  
Работает прямо в браузере — никаких установок не нужно.

## 🚀 Открыть

**GitHub Pages:** https://[username].github.io/trading-app/

Или скачай `index.html` и открой в браузере.

---

## 📱 Вкладки

| # | Вкладка | Что внутри |
|---|---------|------------|
| 🕐 | **Сессии** | Живые часы Монреаль/Токио/Лондон/NY, активная сессия, прогресс, советы |
| 📓 | **Журнал** | Запись сделок, кривая депозита, статистика, винрейт |
| 🚀 | **План** | Путь $300→$10K из журнала, калькулятор цели, этапы, бюджет Strathmore |
| ⚡ | **Инструменты** | Калькулятор зарплаты Квебек, таймер сессии, цель дня |

---

## ✨ Фишки

- **Журнал → План**: «Путь к цели» читает реальные данные из журнала сделок и показывает где ты сейчас на шкале $300→$10K
- Все данные сохраняются в `localStorage` браузера
- Полностью офлайн после первой загрузки
- Оптимизировано под мобильный (max-width 440px)
- Тёмная тема, Binance-стиль

---

## 💾 Установка на GitHub Pages

1. Создай репозиторий (например `trading-app`)
2. Загрузи `index.html`
3. Зайди в **Settings → Pages → Branch: main → Save**
4. Сайт будет на `https://[username].github.io/trading-app/`

---

## 🛠 Стек

- Чистый HTML / CSS / JS — без фреймворков
- Шрифты: DM Sans + Space Mono (Google Fonts)
